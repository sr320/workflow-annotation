How It Works

  1. Submit your sequences: Provide a FASTA file URL or upload a file
  2. Select parameters: Choose sequence type and performance options
  3. Processing: The workflow will:
    • Run BLAST/DIAMOND against Swiss-Prot database
    • Retrieve UniProt annotations via REST API
    • Map GO terms to GO-Slim categories
  4. Download results: Get TSV files with annotations and a summary report
